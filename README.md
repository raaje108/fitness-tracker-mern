// MERN Fitness Tracker
A premium, full-stack fitness application built to help users track their progress, analyze muscle group distribution, and maintain workout consistency.

// Features
Secure Authentication: User accounts protected by JWT (JSON Web Tokens) and password hashing.

Full CRUD Functionality: Create, Read, Update, and Delete workouts with real-time UI updates.

//Smart Visualizations:

Muscle Distribution Pie Chart: Dynamic report showing which muscle groups you're targeting most this month.

Consistency Heatmap: A year-long, GitHub-style activity map to track your gym streaks.

Search & Live Filtering: Instantly find specific exercises or filter by muscle group (Chest, Back, Biceps, etc.).

Midnight Dark Mode: A sleek, system-aware dark theme for low-light environments.

// Tech Stack
Frontend: React.js, Tailwind CSS (Styling), Recharts (Data Viz)

Backend: Node.js, Express.js

Database: MongoDB Atlas (NoSQL)

State Management: React Context API
